How to Install

  1. Download the latest `PomodoroExt.zip` release and extract it to a folder.
  2. Open Google Chrome, and navigate to chrome://extensions/ in your address bar.
  3. Toggle the Developer mode switch in the top-right corner to On.
  4. Click the Load unpacked button in the top-left menu.
  5. Select the extracted PomodoroExt directory.
  6. Click the Puzzle icon in your toolbar and pin PomoFocus for quick access!
